Description
Gloop Glade is a short platformer game where you play as a little slime collecting flowers to make a crown for your mother. 
While traviling throught the glade, the player needs to avoid spiders and large bodies of water to collect all the flowers needed. 
The experience is quick, fun, and relaxing.

To keep things simple, the only major files in the project are the src and assets folders
The src hold all the code, as usual; while the assets folder holds all the art used for the game.
There are a few text files that list the needed libraries as well as the proposal for the game as a project.
