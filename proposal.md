## Description
Gloop Glade will be a cutesy platformer game where you play as a little slime trying to find a flower for your mother. The game is ment to be a fun, relaxing, interactive visual experience.

## Features
- Scrolling screen
	- The screen will move as the player moves; move right, scroll right.
- Movement controlls
	- The player will be able to use either the "A" and "D" keys to move left/right.
- Jumping
	- THe player will be able to use the space bar to jump around the level.

## Challenges
- I will need to learn how to tie the player movement to the screen.
- I will need to learn how to apply basic player controlls to the keyboard.
- I will need to learn how to use and make custom art for a python game.

## Outcomes
Ideal Outcome:
- My ideal outcome is that I make a quick and simple game that a player will have fun with. I want players to be able to relax and not have to worry about super complex game mechanics like puzzles, for example.

Minimal Viable Outcome:
- I want to try and keep this as simple as possible, as what makes a game goos is very subjective. As long as the game is functional and a little bit appealing, that will be a win.
